Para este processo foram utilizadas as linguagens Scala/Java e Scalatest(BDD) em vez de TDD puro.

Requitos Não-Funcionais e testes:
1) Instalar Java 1.8(Última versão disponível se possível).

2) Instalar o Scala 2.11.8: http://www.scala-lang.org/download/

3) Instalar o lightbend-activator-1.3.10 conforme instruções no site: https://www.lightbend.com/activator/download

4) Instalar o ScalaIDE: http://scala-ide.org/download/sdk.html

5) Coloque o diretorio home do lightbend-activator(typesafe-activator) no seu PATH.

6) Crie uma variável que aponte para o repositório do activator no ScalaIDE.

7) Instalar o plugin do Scalatest: https://github.com/scalatest/scalatest-eclipse-plugin

8) Executar os testes conforme as instruções constantes também no item 3 sobre o Spec: AmilGameLog_in_Scala_Spec.scala que está no package com.amil.tests.
(Facílimo mesmo, basta após instalar o plugin do Scalatest mandar executar a classe AmilGameLog_in_Scala_Spec com o target "ScalaTest - File")

9) Caso queira testar outros arquivos é só colocá-los no package: com.amil.resources.In. Atualizar o arquivo de testes com o caminho dos arquivos no mesmo modelo que o atual e reexecutar os testes.
